## 用自己的话回答

- 什么是特征，什么是标签:特征是模型用来做判断的依据。比如客户的“月费”、“合同时长”，这些就是特征。标签是我们希望模型预测出来的结果。“Churn”（是否流失）就是标签。
- 为什么要保留测试集：为了验证真实能力。 如果把所有数据都拿来训练，模型就像背下了考题答案的学生，遇到新题就不会了（这叫过拟合）。保留一部分它没见过的数据作为测试集，才能检验它是不是真的学会了规律。
- 为什么83%准确率仍可能没有用：-因为样本不平衡。 假设数据里有100个人，只有5个人流失了。如果模型什么都不学，直接全部猜“不流失”，它的准确率也有95%。但这毫无意义，因为它根本没抓出那5个要跑的客户。在这种情况下，我们要看的不是准确率，而是召回率。
